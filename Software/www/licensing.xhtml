It is a requirement of some software's licensing that they be given credit, acknowledged, linked or a variation of that.

Therefore all of this projects code is available at git hub: http://github.com/AlistairCheeseman/SmartHome-Project, it lists all project code including script files of the steps taken to create a LFS (linux From Scratch) operating system. Further the EAGLE source files for the design of the hardware (including pdf files for those that do not have access to EAGLE).

The listing is attempted to be kept up to date but for an always up to date listing, please check the GitHub repository.

Software Used in the development

  • Eagle (PCB Design)
  • NetBeans (C++, C, java, CSS, HTML, PHP, XHTML)

Software Used in the ARM Toolchain

Software was built using KMX Software's Build Servers, Bash scripts were written to ensure the repeatability and documentation of the steps taken, also available at the github repostiory.

Some software is included in the OS Software instead of under the Toolchain programs. This was due to the files being specifically for the Smart Home OS rather than a generic OS.
